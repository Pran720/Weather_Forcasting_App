# Weather_Forcasting_App
A modern and fully responsive Weather Forecasting Web Application built using HTML, CSS, and JavaScript.
The app fetches real-time weather data using the OpenWeatherMap API and displays animated backgrounds based on weather conditions.
Features:
Search weather by city name
Real-time temperature display
Humidity and wind speed
5-Day forecast
Dark / Light mode toggle
Animated weather backgrounds
Loading animation
Fully responsive (Mobile-friendly)
Error handling for invalid cities
Technologies Used:
HTML5
CSS3 (Glassmorphism + Animations)
JavaScript (ES6)
OpenWeatherMap API
Project Structure:
Weather-App/
├── index.html
├── style.css
├── script.js
└── README.md
How It Works:
User enters a city name
JavaScript fetches weather data from API
UI updates dynamically
Background changes based on weather condition
Loader appears during data fetching
